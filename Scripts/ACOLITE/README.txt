ACOLITE README

All dependencies are installed under the "acolite" environment so need to launch jupyter notebooks from acolite enviroment:

Launch Anaconda prompt:

>activate acolite
>jupyter notebook

open jupyter notebook script file

If you are using this script need to install everything following these instructions:
https://github.com/acolite/acolite

and create an account here:
https://dataspace.copernicus.eu/

and save credentials in acolite config file
